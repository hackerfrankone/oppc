Understanding chmod Permissions

The chmod command in Linux changes file or directory permissions. Permissions define who can read, write, or execute a file or directory.

Permission Types

Permissions are divided into three categories:

  • Owner (u): The user who owns the file.
  • Group (g): Users in the file's group.
  • Others (o): All other users.

Permission Levels

Each category can have these permissions:

  • Read (r): Permission to read the file or list directory contents (value: 4).
  • Write (w): Permission to modify the file or create/delete files in a directory (value: 2).
  • Execute (x): Permission to run the file (example, scripts) or access a directory (value: 1).

Numeric Notation

Permissions are often set using a three-digit octal number (example, 754):

  • First digit: Owner permissions
  • Second digit: Group permissions
  • Third digit: Others permissions

Each digit is the sum of permission values:

  • 4 (read) + 2 (write) + 1 (execute) = 7 (rwx)
  • 4 (read) + 1 (execute) = 5 (r-x)
  • 4 (read) = 4 (r--)
  • 0 = no permissions (---)

Example: chmod 754 file sets:

  • Owner: rwx (7)
  • Group: r-x (5)
  • Others: r-- (4)

Symbolic Notation

You can also use letters to set permissions:

  • u (owner), g (group), o (others), or a (all).
  • Operators: + (add), - (remove), = (set exactly).
  • Permissions: r, w, x.

Example: chmod u+rwx,g+rx,o+r file sets the same as chmod 754 file.

Directory Permissions

Directories use the same permissions, but:

  • Read: List contents.
  • Write: Create, delete, or rename files in the directory.
  • Execute: Enter the directory or access its contents.

Example: chmod 555 directory allows read and execute for all but no write.

Common Examples

  • chmod 777 file: Full permissions for all (rwx for owner, group, others).
  • chmod 644 file: Owner can read/write (rw-), group and others can read (r--).
  • chmod 755 directory: Owner has full access, group and others can read and enter.

Viewing Permissions

Use ls -l to see permissions, displayed as -rwxr-xr-x or drwxr-xr-x (for directories).
