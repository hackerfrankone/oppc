Critical to Know

  • 20/21 (FTP): File Transfer Protocol. Port 20 for data transfer, 21 for control. Used for transferring files between systems, but unencrypted.
  • 22 (SSH): Secure Shell. Provides secure remote access to systems over an encrypted connection, often used for managing network devices.
  • 23 (Telnet): Unencrypted remote access protocol. Used for terminal access to devices, but largely replaced by SSH due to security concerns.
  • 25 (SMTP): Simple Mail Transfer Protocol. Used for sending emails from a client to a server or between servers.
  • 53 (DNS): Domain Name System. Resolves domain names to IP addresses, essential for internet and network navigation.
  • 80 (HTTP): Hypertext Transfer Protocol. Used for unencrypted web traffic to access websites.
  • 110 (POP3): Post Office Protocol v3. Retrieves emails from a server, commonly used in email clients.
  • 143 (IMAP): Internet Message Access Protocol. Manages and retrieves emails on a server, supporting folder synchronization.
  • 443 (HTTPS): HTTP Secure. Encrypted web traffic using SSL/TLS, standard for secure websites.
  • 3389 (RDP): Remote Desktop Protocol. Enables remote desktop connections to Windows systems for remote management.
