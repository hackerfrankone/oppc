Common Ports

  • 20/21 (FTP): File Transfer Protocol. Port 20 for data transfer, 21 for control. Used for transferring files, but unencrypted and less secure.
  • 22 (SSH): Secure Shell. Used for secure remote access and management of systems over an encrypted connection.
  • 23 (Telnet): Unencrypted remote access protocol. Rarely used due to security risks but may appear in legacy systems.
  • 25 (SMTP): Simple Mail Transfer Protocol. Used for sending emails, often paired with SMTPS for secure transmission.
  • 53 (DNS): Domain Name System. Resolves domain names to IP addresses, critical for network communication.
  • 80 (HTTP): Hypertext Transfer Protocol. Used for unencrypted web traffic, vulnerable to interception.
  • 110 (POP3): Post Office Protocol v3. Retrieves emails from a server, typically unencrypted unless paired with SSL/TLS.
  • 143 (IMAP): Internet Message Access Protocol. Retrieves and manages emails on a server, supports folder synchronization.
  • 443 (HTTPS): HTTP Secure. Encrypted web traffic using SSL/TLS, standard for secure websites.
  • 445 (SMB): Server Message Block. Used for file sharing and printer sharing in Windows environments, often targeted by attacks.
  • 3389 (RDP): Remote Desktop Protocol. Enables remote desktop connections to Windows systems, requires strong security to prevent unauthorized access.
  • 137-139 (NetBIOS): Network Basic Input/Output System. Used for legacy Windows networking services like file and printer sharing.
  • 161/162 (SNMP): Simple Network Management Protocol. Used for monitoring and managing network devices; 161 for queries, 162 for traps.
  • 389 (LDAP): Lightweight Directory Access Protocol. Manages directory services, such as user authentication in Active Directory.
  • 636 (LDAPS): LDAP Secure. Encrypted version of LDAP using SSL/TLS for secure directory access.
  • 993 (IMAPS): IMAP Secure. Encrypted email retrieval using SSL/TLS for secure access to mail servers.
  • 995 (POP3S): POP3 Secure. Encrypted email retrieval using SSL/TLS, enhancing security for email access.
  • 1812/1813 (RADIUS): Remote Authentication Dial-In User Service. Used for authentication, authorization, and accounting in network access.
  • 123 (NTP): Network Time Protocol. Synchronizes clocks across systems, critical for time-sensitive operations like logging.
  • 514 (Syslog): System Logging Protocol. Collects and stores log messages for system monitoring and auditing.
  • 2049 (NFS): Network File System. Enables file sharing across Unix/Linux systems, requires careful configuration for security.
  • 3268/3269 (Global Catalog): Used by Active Directory for global catalog queries (3268) and secure queries over SSL (3269).
  • 1433 (MSSQL): Microsoft SQL Server. Used for database communication, often targeted in attacks if not secured.
  • 3306 (MySQL): MySQL Database. Default port for MySQL database connections, needs protection against unauthorized access.
  • 5432 (PostgreSQL): PostgreSQL Database. Default port for PostgreSQL database connections, requires secure configuration.
  • 1723 (PPTP): Point-to-Point Tunneling Protocol. Used for VPN connections, but considered less secure than modern alternatives.
  • 1701 (L2TP): Layer 2 Tunneling Protocol. Used for VPNs, often paired with IPsec for encryption.
  • 500 (IPsec): Internet Protocol Security. Used for secure VPN connections and encrypted communications.
  • 1194 (OpenVPN): OpenVPN Protocol. Commonly used for secure, open-source VPN connections, highly configurable.
