Tower Chassis

Full-size desktop chassis, supports ATX motherboards, multiple drives, and expansion cards.

Mini-ITX Chassis

Compact chassis for Mini-ITX motherboards, ideal for small form factor PCs.

SSD (Solid State Drive)

Flash-based storage, faster than HDD, used for OS and applications.

HDD (Hard Disk Drive)

Magnetic storage, higher capacity, slower than SSD, used for mass storage.

Optical Mouse

Uses light to track movement, common input device for PCs.

Laser Mouse

Uses laser for precise tracking, suitable for high-DPI tasks.

Mechanical Keyboard

Uses individual switches per key, durable, tactile feedback.

Membrane Keyboard

Uses pressure pads, quieter, commonly used in laptops and budget PCs.

LCD Monitor

Liquid Crystal Display, common, energy-efficient, slim design.

OLED Monitor

Organic LED, superior color and contrast, used in high-end displays.

LED Monitor

Backlit LCD using LEDs, brighter, more energy-efficient than traditional LCD.

CPU (Central Processing Unit)

Processes instructions, brain of the computer, fits specific motherboard sockets.

GPU (Graphics Processing Unit)

Handles graphics rendering, essential for gaming and video editing.

DIMM RAM

Desktop memory, physically larger, used in PCs for temporary data storage.

SO-DIMM RAM

Smaller RAM for laptops, used for temporary data storage.

ATX Motherboard

Standard size, supports multiple expansion slots and components.

Micro-ATX Motherboard

Smaller than ATX, fewer slots, used in compact systems.

Power Supply Unit (PSU)

Converts AC to DC, powers all PC components.

Cooling Fan

Air-based cooling for CPUs, GPUs, or chassis to prevent overheating.

Liquid Cooling

Uses liquid to dissipate heat, efficient for high-performance systems.
